dQ/dt = α − k·Q(t),  Q(0) = Q₀

Balanço de massa

Entrada: r·cin = α [kg/h]
Saída: (r/V)·Q(t) = k·Q [kg/h]
EDO: dQ/dt = α − k·Q

Solução analítica

Q(t) = Q* + (Q₀ − Q*)·e−kt

Q(t) = 20·(1 − e⁻⁰·⁵ᵗ)

Método de Euler

Qn+1 = Qn + h·f(tₙ,Qₙ)
f(t,Q) = α − k·Q
Erro local: O(h²)
Erro global: O(h)

Método RK4

k₁=f(tₙ,Qₙ)
k₂=f(tₙ+h/2, Qₙ+h·k₁/2)
k₃=f(tₙ+h/2, Qₙ+h·k₂/2)
k₄=f(tₙ+h, Qₙ+h·k₃)
Qn+1=Qₙ+h/6·(k₁+2k₂+2k₃+k₄)
Erro global: O(h⁴)

Verificação dimensional

dQ/dt → [kg/h]
α = r·cin → [m³/h]·[kg/m³] = [kg/h] ✓
k·Q → [h⁻¹]·[kg] = [kg/h] ✓

Meia-vida

t½ = ln(2)/k = ln(2)/0.500 ≈ 1.39 h
